## 🎨 Visual Hierarchy

```
SETTINGS (Exploration)
    └─ Blueprint Explorer
         └─ Learn about all 25 blueprints

CREATION (Wizard)
    └─ Enhanced Wizard
         ├─ Step 1: Idea
         ├─ Step 2: Goal & Audience ← Uses Audience Helper
         ├─ Step 3: Platform
         ├─ Step 4: Length
         └─ Step 5: Blueprint

DRAFTING (Editor)
    ├─ Formatting Toolbar
    │    └─ Remix Tool (modal)
    └─ Jamie Panel
         ├─ Section 1: Blueprint Template
         ├─ Section 2: Tone & Voice
         ├─ Section 3: Audience Guidance ← Audience Helper
         ├─ Section 4: SEO & Metadata
         └─ Section 5: Review & Polish ← CTA Suggester

PUBLISHING (Post-Publish)
    └─ Repurposing Modal
         └─ Suggest 2 repurposing ideas
              └─ Create new drafts
```

## 💡 User Mental Model

```
Before writing:
  "I need to browse blueprints to understand my options"
  → Blueprint Explorer

Starting new content:
  "Who am I writing for and why?"
  → Enhanced Wizard (Goal & Audience step)

During drafting:
  "This section needs a different tone"
  → Remix Tool

  "What should I say about this topic for clinicians?"
  → Audience Helper

  "How should I end this post?"
  → CTA Suggester

After publishing:
  "How can I repurpose this content?"
  → Repurposing Modal
```
