1. **Standalone App:** Content is the default view (no URL parameters)
2. **Original Preserved:** All tasks/contacts/calendar work is saved
3. **No Migration:** Continue building in this file with Builder 1
4. **Jamie First:** Focus on making AI assistance excellent
5. **Simple Over Complex:** Table view, not fancy gallery
